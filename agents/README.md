# Agents Index

Agents orchestrate skills and tools to accomplish complex, multi-step objectives. Each agent defines what it does, which skills/tools it uses, and how to handle edge cases.

| Agent | Description | Skills Used |
|-------|-------------|-------------|
| [Outreach Agent](outreach_agent.md) | Research leads → draft emails → manage campaigns | Lead Management, Pipeline Operations, Email Campaign Analytics |
| [Analytics Agent](analytics_agent.md) | Pull data from all sources → generate insights | Email Campaign Analytics, CRM Reporting |
| [Lead Enrichment Agent](lead_enrichment_agent.md) | Enrich leads with Apollo API data | Lead Management |
| [Pipeline Manager Agent](pipeline_manager_agent.md) | Monitor pipeline health, suggest follow-ups | Pipeline Operations, CRM Reporting, Lead Management |
